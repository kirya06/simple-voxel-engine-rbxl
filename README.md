## simple roblox voxel engine

fully server-side voxel engine using parts
https://www.roblox.com/games/85722289133953/voxel-engine-using-parts

---

Little project that I started last weekend. At first I wanted to use the new "EditableMesh" class... but it couldn't even create 3 chunk meshes at the same time because of memory limits. Maybe it's intented or maybe it's some kinda linux compatability issue, but either way because of that I decided to make a dumb dumb server-side voxel engine using only parts.
